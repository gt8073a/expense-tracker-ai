# expense-tracker-ai

A modern expense tracking web application built as part of a Claude Code class project.

## About

This app was built using [Claude Code](https://claude.ai/code) to demonstrate how AI-assisted development can rapidly produce a full-featured, production-ready web application.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- Data persistence via `localStorage`

## Features

- Add, edit, and delete expenses with date, amount, category, and description
- Dashboard with spending summaries (total, monthly, weekly) and category breakdown chart
- Filter and search expenses by keyword, category, and date range
- Export expenses to CSV
- Responsive design — works on desktop and mobile

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
